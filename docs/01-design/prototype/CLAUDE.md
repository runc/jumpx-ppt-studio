# Jumpx Slides — 项目记忆

AI 幻灯片生成 WebApp 的界面外壳(非幻灯片成品)。中文界面,受众=中文训练营学员(非设计师)。核心理念:画布为主、AI 为副驾,刻意避开"紫色渐变 SaaS 聊天框"。

## 设计系统(已定)
- 基调:纸感工作室,暖白象牙纸面,大留白,发丝分隔线,无渐变堆叠。
- 颜色:--paper-0 #FAF8F2(App底) / --paper-1 #FFFFFF(卡片·幻灯片) / --paper-2 #F1EEE4(桌面) / --hair #E4DFD0(分隔线);文字 --ink #23251F / --ink-2 #5E5F54 / --ink-3 #94927F。
- 强调:墨绿 --pine #2C5E48 / --pine-soft #E7EFE8,只在阶段、主操作、AI高亮处克制使用。极少量暖辅色 --amber #B07A2E(进行中/提示)。
- 字体:中文 Noto Sans SC(思源黑体);数字/标签 Space Grotesk;活动流时间戳 Space Mono。
- 阶段流程:输入 → 大纲 → 选模板 → 渲染 → 完成。

## 布局基底(已定 = 工作台方向B + 副驾A)
- 舞台优先:当前页做成"裱框"主角(暖桌面 + 软投影),缩略图为底部横排胶片轨(16:9 需要横向宽度)。
- 右侧副驾走克制版,优先级:① 三态待办(置顶)② 对话框(常驻底部)③ 活动流"它在干什么"(可折叠)。
- 交互:生成中自动展开活动流(高光时刻),完成后自动收起。

## 5 屏方向定案(并排对比稿均已交付)
1. 主工作台 = **B 舞台优先**(裱框主角 + 底部横排胶片轨)+ A 克制副驾。副驾优先级:三态待办置顶 / 对话常驻底部 / 活动流可折叠;**生成中自动展开活动流、完成后自动收起**。文件 Jumpx Slides/主工作台.html
2. 输入起步页 = **A 居中聚焦**("想讲点什么?"为唯一主角,不摆空裱框)+ 例子chips + 篇幅/受众/语气前置设置 + 借B:点开始生成后平滑过渡进工作台(空裱框→骨架→逐页填充)。文案沿用"想讲点什么?""几个关键选择,它会停下来问你的意见"。文件 输入起步页.html
3. 大纲编辑器 = **B 通览(故事板网格)为主视图** + A 单页深编作"双击卡片钻入"二级(与"渲染后改单页"复用同一组件)。大纲阶段保持轻:改标题/一句话/拖拽调序/增删/采纳建议。保留可拖拽树 + 副驾建议加页✓/✗ + 确认CTA + 重拟/调节奏。文件 大纲编辑器.html
4. 选模板 = **B 套封面预览为基底**,左侧7缩略图渲染"你真实封面在该风格下"的样子(非抽象色条,同套cqw代码)+ 右大图确认;保留副驾推荐预选+角标+理由。7套:素白(推荐)/学术绿/牛皮纸/夜读/杂志/图表报告/圆润。文件 选模板.html
5. 选输出形态 = **B 同页两版为基底**(同一页并排渲 HTML / AI配图,show别只tell)+ A 诚实取舍(每版下带速度/可编辑/视觉标签+关键代价,AI明说"较慢·不易再改")。副驾推荐HTML预选+理由+顶栏"用HTML生成·开始→"。**AI配图预览只用示意样图,不真生成**。文件 选输出形态.html
