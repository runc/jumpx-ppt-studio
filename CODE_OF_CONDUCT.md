# 贡献者行为准则 · Code of Conduct

本项目采用 [Contributor Covenant](https://www.contributor-covenant.org) v2.1 作为社区行为准则。

## 我们的承诺

为营造开放、友善、多元、无骚扰的环境，我们承诺：无论年龄、体型、可见或不可见的残疾、族裔、性别认同与表达、经验水平、教育、社会经济地位、国籍、外貌、种族、宗教或性取向如何，参与本项目的每个人都应受到尊重。

## 我们的标准

有助于营造良好环境的行为包括：

- 对他人表达善意与同理；
- 尊重不同的观点、视角与经验；
- 优雅地给出并接受建设性反馈；
- 为自己的错误负责、向受影响者致歉并从中学习；
- 以整个社区的长远利益为重。

不可接受的行为包括：

- 性化的言语或图像、以及任何形式的性骚扰或挑逗；
- 钓鱼、侮辱或贬损性评论，人身或政治攻击；
- 公开或私下的骚扰；
- 未经明确许可公开他人隐私信息（如住址、邮箱）；
- 其他在专业场合中合理判断为不当的行为。

## 责任与执行

项目维护者有权移除、编辑或拒绝不符合本准则的评论、提交、代码、issue 及其他贡献，并在必要时对违规者采取临时或永久封禁。

如遇违规行为，请通过 **GitHub Issue** 或私信 [JumpX Labs](https://github.com/JumpX-Labs) 上报 <!-- ★ 可补一个联系邮箱 -->。所有投诉都会被审阅与调查，并会尊重举报者的隐私与安全。
